## Currently Building

- **[Dough](https://github.com/dough-platform)** — An open-source agent orchestration platform that runs each agent in its own Firecracker MicroVM for secure, lightweight isolation.
- **Reactive Event Bus** — A CEL-based rules engine that lets agents communicate and trigger workflows through declarative, event-driven policies.
- **[Pad](https://github.com/dough-platform/pad)** — An agent-native task manager designed for humans and AI agents to collaborate on planning, tracking, and executing work.
